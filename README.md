📊 Gen-Z Dating Apps Market Analysis using Power BI
📌 Project Overview

This project analyzes the behavior, preferences, engagement, and satisfaction levels of Gen-Z users across popular dating applications using Power BI. The dashboard provides actionable insights into app popularity, user engagement patterns, ratings, and demographic trends through interactive visualizations and KPI tracking.

🎯 Objectives
Analyze the popularity of dating apps among Gen-Z users.
Understand user engagement and daily usage patterns.
Compare app ratings and user satisfaction levels.
Identify demographic trends based on age and gender.
Generate business insights and recommendations using data-driven analysis.

🛠️ Tools & Technologies
Power BI Desktop
Power Query (Data Cleaning & Transformation)

CSV Dataset

🔄 Data Preparation
Handled missing values using Power Query.
Standardized categorical data.
Converted daily usage time from text format (e.g., "1.5 hours", "30 minutes") into numerical minutes.
Created calculated columns for:
Engagement Level
Rating Level
Age Segmentation

📊 Key Insights
Identified the most preferred dating apps among Gen-Z users.
Analyzed the relationship between engagement levels and user ratings.
Evaluated satisfaction trends across different applications.
Explored behavioral patterns based on demographic segments.

🚀 Business Recommendations
Focus on features that improve user engagement and retention.
Enhance user experience in lower-rated applications.
Leverage demographic insights for targeted marketing strategies.
Optimize app functionality based on user feedback and challenges.
